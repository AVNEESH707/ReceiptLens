# 🚀 ReceiptLens – AI Receipt OCR & Parser

ReceiptLens is a full-stack web application that extracts and structures data from receipt images using OCR.

🔗 Live Demo  
Frontend: https://receipt-lens-32qhmz9or-avneesh707s-projects.vercel.app/  
Backend API: https://receiptlens.onrender.com  

---

## ✨ Features

- 📸 Upload receipt images
- 🔍 OCR using Tesseract.js
- 🧾 Extract:
  - Store Name
  - Date
  - Total Amount
  - Items with price
- ⚡ Fast REST API (Node.js + Express)
- 🎯 Smart parsing logic for messy OCR output

---

## 🛠️ Tech Stack

### Frontend
- React.js
- Tailwind CSS
- Axios

### Backend
- Node.js
- Express.js
- Tesseract.js (OCR)
- Multer (file upload)
